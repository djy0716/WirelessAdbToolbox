# Changelog

## 1.1.0 — 2026-07-21

- Fixed content being drawn underneath the status bar and bottom gesture navigation area.
- Added IME-aware WindowInsets handling.
- Replaced the long single-page layout with Connection, APK, Shell, Device and Log tabs.
- Added a fixed connection-status card.
- Separated Android 11+ pairing and connection workflows.
- Added push-only and push-and-install APK actions.
- Added optional cleanup of the remote temporary APK after installation.
- Added pause, copy, save and clear controls for logs.
- Added APK file name/size and operation status display.
